Javathis和super关键字

this

  • 调用当前类的方法；
  • this()  可以调用当前类的构造方法；
  • this 可以作为参数在方法中传递；
  • this 可以作为参数在构造方法中传递；
  • this 可以作为方法的返回值，返回当前类的对象

super

  • 指向父类对象；
  • 调用父类的方法；
  • super()  可以调用父类的构造方法。
